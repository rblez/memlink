# Installation

## Via npm

```bash
npm install -g @memlink/cli
```

## Via pnpm

```bash
pnpm install -g @memlink/cli
```

## Via Yarn

```bash
yarn global add @memlink/cli
```

## Via Bun

```bash
bun install -g @memlink/cli
```

## From source

```bash
git clone https://github.com/aiustantdotdev/memlink.git
cd memlink
bun install
npm run build
```

The binary is then available as `memlink` from the command line.

## Verify installation

```bash
memlink --version
```
