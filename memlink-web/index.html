Why Memlink

One memory store, one URL. No tokens, no setup, no cloud.

Zero Config

Install and run. No tokens, no API keys, no cloud accounts. One URL connects any agent.

Any Agent

Works with Claude, Cursor, Windsurf, Codex, OpenCode, Cline, and any MCP-compatible tool.

Local-First

Self-hosted on your machine. Data never leaves your control. No cloud, no vendor lock-in.

CLI at a Glance

Every command is designed to be simple, predictable, and self-documenting.

Create a Memory

memlink init creates a new memory — name, ID, and MCP URL in one command.

Start the Server

memlink serve boots the MCP server with all your memories. Optional --cors and --read-only.

Connection Details

memlink connect shows the MCP URL and copies it to your clipboard.

WSL Bridge

memlink wsl-connect auto-detects the Windows gateway IP so agents in WSL can connect.
